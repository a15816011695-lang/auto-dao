## 学习类型说明

### 人文社科类
学习哲学、心理学、社会学等领域的书籍，注重概念理解和思想探讨。

### 理科类
学习数学、物理等基础学科教材，注重定理推导和习题练习。

### 工科论文
学习机器学习、计算机科学等领域的技术论文，注重算法理解和实现细节。

### 开源项目
学习 GitHub 上的开源项目，注重代码结构、设计模式和实际应用。
